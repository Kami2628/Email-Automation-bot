# 📧 Email Automation Bot

A simple Gmail Email Automation project using Python + Streamlit.

---

## 🚀 Features

- 🔐 Gmail Login (Google OAuth)
- 📧 Send Emails
- 📎 Attach Photos & Files
- ⏰ Schedule Emails (Date & Time)
- 📜 Email History

---

## 🛠️ How to Run

### 1. Install requirements
```bash
pip install streamlit google-auth-oauthlib google-api-python-client
